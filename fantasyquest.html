Fantasy Quest Project Documentation

About the Game

FantasyQuest (working title) is a dark fantasy rpg. Despite its old-school inclinations, it will be a full-featured rpg including character classes and races, a leveling, loot, combat, and magic system, interactive NPCs, and what I intend to be a gripping narrative. More to come as it comes. Sourcecode can be found on my GitHub page: analystrusso/fantasyquest
